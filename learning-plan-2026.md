## 📌 Важные акценты (от DeepSeek)

### 1️⃣ DOM — не застревай
**Нужно знать:**
- `querySelector`, `getElementById`
- Классы, атрибуты
- События (`addEventListener`, всплытие)

**НЕ нужно углубляться:**
- `createElement`, ручная сборка интерфейсов через DOM
- В React это делается иначе

### 2️⃣ React: сначала классовые, потом хуки
| Неделя | Тема | Зачем |
|--------|------|-------|
| 1-2 | Классовые компоненты | Понять жизненный цикл, `this`, `componentDidMount` |
| 3-4 | Функциональные + хуки | `useState`, `useEffect`, современный стандарт |

**Почему:** на собеседовании могут спросить разницу — будешь готов.

### 3️⃣ База данных: PostgreSQL (приоритет)
**Почему не MongoDB:**
- Яндекс тяготеет к реляционным БД
- PostgreSQL учит правильно проектировать данные
- MongoDB можно добавить позже как «второй язык»

### 4️⃣ Алгоритмы: с 6-го месяца (параллельно)
**Не откладывать на месяц 10!**
- 15-20 минут в выходной на одну задачу LeetCode Easy
- К собеседованию — «натренированный глаз»

**Что спрашивают на внутреннем переходе:**
- REST (как устроен, статусы, методы)
- Базы данных (индексы, нормализация)
- HTTP (запросы, заголовки, куки)
