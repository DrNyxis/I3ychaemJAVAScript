## 📚 Что прошли:

### 1. Основы JavaScript
- Переменные: `const` (нельзя менять), `let` (можно менять)
- Типы данных: string, number, boolean, null, undefined
- Вывод в консоль: `console.log()`

### 2. Работа со страницей (DOM)
- 📦 `document.getElementById('id')` — найти элемент
- 📦 `element.addEventListener('click', function)` — повесить событие
- 📦 `element.textContent` — изменить текст
- 📦 `element.style.color` — изменить цвет
- 📦 `element.style.background` — изменить фон
- 📦 `element.innerHTML` — вставить HTML
- 📦 `element.value` — получить значение из input

### 3. Условия
- `if (условие) { ... } else { ... }`
- Операторы: `===`, `!==`, `>`, `<`, `>=`, `<=`
- Логические: `&&` (И), `||` (ИЛИ), `!` (НЕ)

### 4. ФУНКЦИИ ✅ (17 марта)
- Как создать: `function myFunction() { ... }`
- Параметры: `function add(a, b) { ... }`
- Возврат значения: `return a + b`
- Вызов: `add(2, 3)`
- Скобки: `()` — параметры/вызов, `{}` — тело функции

## 📌 Важные заметки:

- 📦 = встроенное (браузер/JS): document, alert, console.log
- 👤 = твоё (ты создал): title, btn, myName, addF, val1, wishes, addWish
- ID в HTML и JS должны совпадать!
- Скрипт ставь ПОСЛЕ элементов (или в конце body)
- Всегда используй `===` вместо `==`
- `return` возвращает значение из функции
- `()` — параметры/вызов функции, `{}` — тело функции
- `.value` — получить/установить значение из input
- `Number()` — преобразовать строку в число
- `innerHTML +=` — добавить HTML внутрь элемента
- `push()` — добавить элемент в массив
- Кавычки: используем `'` (одинарные) для единообразия
- `;` — хороший тон, ставь в конце инструкций

## 🔧 Типичные ошибки:
- Опечатки: `getElemenById` → `getElementById`
- Несовпадение id в HTML и JS
- Код вне addEventListener (выполняется сразу)
- Забытые скобки и кавычки
- Конфликт имён: функция и переменная одинаково называются
- `num1.value` вместо `num2.value` (вычитал одно из того же)
- Забытый слэш в закрывающем теге: `<li>` → `</li>`
- `addEventListener('click', func)` — функция БЕЗ скобок!
